About LiverCare AI

Understanding our mission, methodology, and technology

Project Overview

LiverCare AI is an advanced machine learning-based system designed to assist in the early detection of liver disease. Our platform leverages state-of-the-art ensemble learning techniques to provide accurate and reliable risk assessments.

Early detection of liver disease is crucial for effective treatment and improved patient outcomes. Our system analyzes multiple health indicators including age, BMI, lifestyle factors, medical history, and liver function test results to provide comprehensive risk assessment.

Methodology

Machine Learning Models

Our system employs multiple machine learning algorithms to ensure robust and accurate predictions:

1. Logistic Regression

Baseline linear model providing interpretable coefficients for feature importance.

2. Random Forest

Ensemble of decision trees capturing non-linear relationships with 96.7% accuracy.

3. XGBoost

Gradient boosting algorithm achieving 96.9% accuracy with excellent feature handling.

4. Support Vector Machine (SVM)

RBF kernel SVM providing 91.1% accuracy with strong generalization.

5. Neural Network (MLP)

Multi-layer perceptron with 128-64 hidden layers achieving 96.9% accuracy.

Ensemble Methods

Voting Classifier (MLP + Random Forest)

Our best-performing model combining MLP and Random Forest through soft voting, achieving 97.9% accuracy and 99.5% ROC-AUC.

Voting Classifier (MLP + XGBoost)

Alternative ensemble achieving 97.5% accuracy with excellent precision.

Stacking Classifier

Meta-learner approach using Logistic Regression to combine predictions from multiple base models.

Input Features

  • Age: Patient age in years
  • Gender: Biological gender (0: Female, 1: Male)
  • BMI: Body Mass Index
  • Alcohol Consumption: Alcohol intake level
  • Smoking: Smoking status (0: No, 1: Yes)
  • Genetic Risk: Family history of liver disease (0: No, 1: Yes)
  • Physical Activity: Level of physical activity
  • Diabetes: Diabetes status (0: No, 1: Yes)
  • Hypertension: High blood pressure status (0: No, 1: Yes)
  • Liver Function Test: Liver function test value
